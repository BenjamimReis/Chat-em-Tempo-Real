# Chat-em-Tempo-Real
Aplicativo de chat com salas, mensagens privadas, notificação em tempo real, autenticação e histórico de mensagens.
# Chat em Tempo Real Profissional

## Tecnologias
- Frontend: React + TypeScript + Redux + TailwindCSS
- Backend: Node.js + Express + Socket.IO + PostgreSQL + MongoDB
- Autenticação: JWT + OAuth
- DevOps: Docker + docker-compose

## Setup
1. Clonar repo
2. Rodar `docker-compose up --build`
3. Frontend: http://localhost:3000
4. Backend: http://localhost:5000

## Funcionalidades
- Salas de chat em tempo real
- Mensagens privadas e públicas
- Histórico de mensagens
- Notificações em tempo real
- Sistema multi-usuário
- Deploy pronto para Kubernetes ou AWS
